
# Arithmetic Formatter

## Description
Formats arithmetic expressions vertically. 

```
problems = ["32 + 8", "1 - 3801", "9999 + 9999", "523 - 49"]
output = 

   32         1      9999      523
 +  8    - 3801    + 9999    -  49
 ----    ------    ------    -----
   40     -3800     19998      474
```

## Purpose
To review python to follow CSE 312.
